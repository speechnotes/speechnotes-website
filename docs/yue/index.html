修整標點及多餘空白

請確認是否要修整已輸入的標點及廣東字，並移除中文字之間的多餘空白？

把半形標點改為全形標點，並把所有用 "o" 代表廣東字的字詞修正為單一個字，例如：

「o係」→「喺」
「o的」→「啲」
「o野」→「嘢」
「o黎」→「嚟」

按：一經執行後無法還原 (undo)

  • 按右上方的 啟動或暫停 Speechnotes 語音輸入文字，按右方的符號鍵插入標點符號。
  • 按上方的【常用廣東字】顯示選項鍵，即「哋」、「喺」、「啲」、「嚟」、「咗」、「嘅」、「嗰」、「嘢」、「吓」、「攞」，再按一下選項鍵把它插入內文。

    • 按 D 會出現「哋」、「啲」、「唞 」、「嘟」、「兜」等選項
    • 按 H 會出現「吓」、「慳」、「喺 」、「冚」、「嚡」等選項
  • 你也可以用傳統 o+ 的方式輸入廣東字，例如：
    • 輸入「o的」自動轉 →「啲」
    • 輸入「o地」自動轉 →「哋」
    • 輸入「o既」自動轉 →「嘅」
    • 輸入「o下」自動轉 →「吓」
    • 輸入「o左」自動轉 →「咗」
    • 輸入「o黎」自動轉 →「嚟」......如此類推。
